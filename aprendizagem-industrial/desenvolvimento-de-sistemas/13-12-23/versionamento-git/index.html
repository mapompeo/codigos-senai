Começando em Versionamento

  • Para visualizar a versão atual do GIT, basta usar git --version
  • Para iniciar o GIT, basta usar git init
  • Para visualizar os arquivos do diretório, basta usar ls
  • Para visualizar os atributos ocultos, basta usar ls -a
  • Para avançar, basta usar cd ..
  • Para voltar, basta usar cd ../..
  • Para limpar a tela, basta usar clear
  • Para usar os comandos anteriores, basta usar seta para cima
  • Para visualizar o usuário do Git Bash, basta usar git config --global user.name
  • Para inserir o usuário do Git Bash, basta usar git config --global user.name "seu nome"
  • Para visualizar o e-mail do usuário do Git Bash, basta usar git config --global user.email
  • Para inserir o e-mail do usuário do Git Bash, basta usar git config --global user.email "seu-email@email.com"

  • Para inserir um arquivo, basta usar git add "seu arquivo"
  • Para inserir todos os arquivos, basta usar git add ./
  • Para ver o status dos arquivos, basta usar git status
  • Para fazer um commit e escrever uma mensagem, basta usar git commit -m "sua mensagem"
  • Para visualizar as alterações realizadas, basta usar git log
